# 記憶模型與學習適應性模擬之實作

## Implementation of Memory Model and Learning Adaptability Simulation

## 作業主題

> **What is the KEY in Learning?**

本程式以強化學習（Reinforcement Learning）模擬「記憶長短」對學習行為的影響，並量化三項指標：**適應時間**、**感恩分數**、**固執分數**，藉此說明記憶太短或太長各自帶來的問題。

---

## 核心概念

環境在 t = 120 步時發生改變：

| 時期 | 選擇 A（舊） | 選擇 B（新） |
|------|------------|------------|
| 改變前 | 獎勵機率 80% | 獎勵機率 20% |
| 改變後 | 獎勵機率 20% | 獎勵機率 80% |

Agent 必須在環境改變後，學會放棄舊的好選擇（A），轉向新的好選擇（B）。

---

## 三項評量指標

### 1. 適應時間（Adapt Time）
- **定義**：環境改變後，Agent 在連續 20 步內有 ≥ 80% 的時間選擇 B，所花費的步數。
- **意義**：數字越小，代表適應越快。

### 2. 感恩分數（Gratitude Score）
- **定義**：環境改變後**早期**（前 30 步）仍選擇舊選擇 A 的比例（%）。
- **意義**：數字越低，代表越快忘記過去的恩人（舊好選擇）→ **忘恩負義（短期記憶特徵）**。

### 3. 固執分數（Stubborn Score）
- **定義**：環境改變後**後期**（第 120～180 步）仍堅持選擇 A 的比例（%）。
- **意義**：數字越高，代表難以放棄舊習慣 → **固執（長期記憶特徵）**。

## 兩種記憶模式比較

| 參數 | 短期記憶 | 長期記憶 |
|------|---------|---------|
| 學習率 α | 0.55（快速更新） | 0.12（緩慢更新） |
| 遺忘率 | 0.12（快速衰減） | 0.01（緩慢衰減） |
| 探索率 ε | 0.15 | 0.08 |

### 短期記憶（Short Memory）
- ✅ 適應快（Adapt Time 較小）
- ❌ 很快忘記舊的好選擇 → **感恩分數低（忘恩負義）**

### 長期記憶（Long Memory）
- ✅ 感恩分數高（記得舊恩人）
- ❌ 環境改變後難以調整 → **固執分數高（固執）**

---

## 模型核心：Agent 更新規則

每一步的更新流程：

```
1. 兩個選項的 Q 值同步衰減（遺忘）：
   Q[A] *= (1 - forgetRate)
   Q[B] *= (1 - forgetRate)

2. 對本次選擇的 Q 值做學習更新：
   Q[action] += α × (reward - Q[action])
```

選擇策略採用 **ε-greedy**：
- 機率 ε：隨機探索
- 機率 1-ε：選擇 Q 值最高的動作

## 模擬參數設定

| 常數 | 數值 | 說明 |
|------|------|------|
| `TOTAL_STEPS` | 300 | 總步數 |
| `CHANGE_STEP` | 120 | 環境改變的時間點 |
| `RUNS` | 500 | 每組實驗重複次數 |
| `GRATITUDE_WINDOW` | 30 | 感恩分數計算窗口 |
| `STUBBORN_START` | 120 | 固執分數開始步數（相對改變後） |
| `STUBBORN_WINDOW` | 60 | 固執分數計算窗口 |
| `ADAPT_WINDOW` | 20 | 適應時間判斷窗口 |
| `ADAPT_THRESHOLD` | 0.80 | 適應判斷門檻（80% 選 B） |

## 結論

學習的關鍵在於**記憶長度的平衡**：

- 記憶**太短** → 快速適應新環境，但容易忘記過去的經驗（忘恩負義）
- 記憶**太長** → 能保留過去的好經驗，但難以因應環境改變（固執）

最佳學習策略需要在「遺忘舊知識」與「保留有價值的經驗」之間取得平衡。
